Features

Alles was du für den DHCP-Alltag brauchst – direkt im Browser.

📊
Dashboard
KPI-Cards mit aktiven Leases, statischen Einträgen, Pool-Größe und Leasetime – auf einen Blick.
📋
Aktive Leases
Echtzeit-Tabelle aller DHCP-Leases aus der PiHole API mit Suchfilter und Schnell-Fix als statisch.
📌
Statische Einträge
MAC → IP Bindungen hinzufügen und löschen – PiHole wird automatisch neu geladen.
⚙️
Konfiguration
DHCP-Pool, Gateway, DNS-Server und Leasetime direkt über die Web-UI bearbeiten.
🔄
Auto-Session
Login per PiHole-Passwort. Session-Token wird gecacht und bei Ablauf automatisch erneuert.
🐳
Portainer-Ready
Einzelner Stack, kein externes Netzwerk, keine Volumes nötig – einfach deployen.
🌙
Dark / Light Mode
System-aware Theme, manuell umschaltbar. Sieht auf jedem Gerät gut aus.
🔒
Secret Scanning
TruffleHog scannt automatisch bei jedem Push auf Secrets und blockiert den Merge bei Fund.

Schnellstart

In 3 Schritten zum laufenden Dashboard.

1
Repository klonen
git clone https://github.com/jbkunama1/hAI.PeeDHCP.git
cd hAI.PeeDHCP
2
.env anlegen & anpassen
cp .env.example .env
# Zwei Werte eintragen:
PIHOLE_URL=http://192.168.178.1
PIHOLE_PASSWORD=dein-pihole-passwort

⚠️ Passwort wird nur zum Einloggen verwendet – nie ins Repo committen.

3
Starten & aufrufen
docker compose up -d --build
# Dashboard unter:
http://<server-ip>:8095
